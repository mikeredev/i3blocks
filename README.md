#### Requirements
`alsa-utils`
`fontawesome`
`nmcli`
`nvidia`
`psutil`

#### Installation
- Copy or clone this repository as `$HOME/.config/i3blocks`
- Make the control script executable via `chmod +x ~/.config/i3blocks/i3blocks.py`
- Start via i3 `status_command i3blocks -c ~/.config/i3blocks/i3blocks.conf`
